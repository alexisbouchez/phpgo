Welcome to the PHPGo Development Server

This server demonstrates PHPGo's ability to execute PHP code, including WordPress-compatible functionality.

Features

  • Support for superglobals ($_GET, $_POST, $_SERVER, etc.)
  • Complete OOP support (classes, inheritance, interfaces)
  • WordPress-style plugin and theme functionality
  • Database operations (MySQLi)
  • JSON API support (REST API compatible)
  • Session and cookie handling
  • Error handling and debugging
