NOTES
==============================================
- WordPress requires PHP 7.4+ (recommend targeting PHP 8.0+)
- Core WordPress can run with limited extensions if database + HTTP work
- Many WordPress plugins require additional extensions (GD, cURL, etc.)
- REST API requires JSON (already supported ✓)
- Block editor (Gutenberg) heavily uses REST API
- Consider implementing most critical features first, then expand
- Some features like cURL can be substituted with stream wrappers
